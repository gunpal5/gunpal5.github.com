      TuningTask Properties

      Properties

      CompleteTime Output only. The timestamp when tuning this model completed. Uses RFC 3339, where generated output will always be Z-normalized and uses 0, 3, 6 or 9 fractional digits. Offsets other than "Z" are also accepted. Examples: "2014-10-02T15:01:23Z", "2014-10-02T15:01:23.045123456Z" or "2014-10-02T15:01:23+05:30".
      Hyperparameters Immutable. Hyperparameters controlling the tuning process. If not provided, default values will be used.
      Snapshots Output only. Metrics collected during tuning.
      StartTime Output only. The timestamp when tuning this model started. Uses RFC 3339, where generated output will always be Z-normalized and uses 0, 3, 6 or 9 fractional digits. Offsets other than "Z" are also accepted. Examples: "2014-10-02T15:01:23Z", "2014-10-02T15:01:23.045123456Z" or "2014-10-02T15:01:23+05:30".
      TrainingData Required. Input only. Immutable. The model training data.
